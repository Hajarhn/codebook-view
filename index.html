About this codebook
PCNN variable dictionary — researcher viewer v1.1

The PCNN Codebook is the variable dictionary for the Post-COVID Network Netherlands, a national collaborative research initiative funded by ZonMw. It documents the validated questionnaires used across PCNN sub-studies (PROMIS-29, DSQ family, EQ-5D-5L, COMPASS-31, Pain DETECT, SF-36, and others) and aims to support FAIR (Findable, Accessible, Interoperable, Reusable) data alignment across the network.

This viewer is a read-only, self-contained copy of the codebook intended for principal investigators and study staff. Editing is done by the codebook owner in a separate editor; updated viewers are re-published as the codebook evolves.

The codebook data lives in codebook.json next to this HTML on GitHub Pages. When the codebook is updated, only codebook.json changes — this HTML stays the same. Refresh the page to see the latest version.
